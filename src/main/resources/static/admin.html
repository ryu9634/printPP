폰트 설정

Google Fonts에서 원하는 폰트를 선택 → "Get embed code" 클릭 → "@import" 탭의 URL을 복사하여 "폰트 URL"에 붙여넣고, font-family 이름을 입력하세요.
예시) URL: https://fonts.googleapis.com/css2?family=Playfair+Display&display=swap
